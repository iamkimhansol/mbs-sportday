# 노래 추천 웹사이트 설정 가이드

웹사이트를 정상적으로 작동시키려면 **Firebase 설정**이 필요합니다. (음악 검색은 별도의 키 없이 iTunes API를 사용합니다.)

## 1. 음악 검색 (iTunes API)
- 현재 음악 검색은 별도의 API 키 발급 없이 무료로 작동합니다. 
- 검색 시 정식 음원 데이터(곡명, 아티스트, 앨범 아트)가 깔끔하게 표시됩니다.
- 추천 목록에서 **YouTube 아이콘**을 클릭하면 해당 곡의 검색 결과로 바로 연결됩니다.

## 2. Firebase 설정
1. [Firebase Console](https://console.firebase.google.com/)에 접속합니다.
2. 새 프로젝트를 추가합니다.
3. **Firestore Database** 메뉴에서 **데이터베이스 만들기**를 클릭합니다. (위치는 가급적 가까운 곳, 모드는 **테스트 모드**로 시작하는 것이 편합니다.)
4. 프로젝트 개요 페이지에서 **웹 앱(</>)** 아이콘을 클릭하여 앱을 등록합니다.
5. 제공되는 `firebaseConfig` 객체의 값들을 `.env.local` 파일의 해당 항목들에 각각 복사해 넣습니다.

## 3. 실행 방법
프로젝트 폴더에서 다음 명령어를 입력하세요:
```bash
npm run dev
```
이후 `http://localhost:3000`에서 확인하실 수 있습니다.
- **관리자 페이지**: `http://localhost:3000/admin`
- **초기 비밀번호**: `1234`

## 4. 주요 기능
- **노래 검색**: YouTube 실시간 검색 연동
- **중복 방지**: 이미 추가된 노래는 다시 추가할 수 없음
- **실시간 목록**: 누군가 노래를 추가하면 모든 사용자의 화면에 즉시 나타남 (Firebase Firestore)
- **관리자 모드**: 개별 삭제 및 전체 초기화 가능
